## Code Changes

### Feed bug fixes (`src/cli/feed.ts`, `src/specialist/timeline-query.ts`)
Three correctness fixes: (1) seed pass completion now prints "All jobs complete." to stderr instead of silently exiting; (2) empty jobs dir now prints "No jobs found." instead of "All jobs complete."; (3) `feed <job-id>` now tails the latest N events (`slice(-limit)`) instead of replaying from start (`slice(0, limit)`). Tests updated to assert latest timestamps.

### TMUX cwd fix (`src/pi/session.ts`)
Single-line fix: `cwd: resolve(this.options.cwd ?? process.cwd())` in Pi spawn options. Pins absolute path at spawn time so Pi file operations always resolve to the correct worktree regardless of TMUX relative-path behavior. New tests in `tests/unit/pi/session.test.ts` assert default and relative cwd both produce absolute paths.

### specialists list improvements (`src/cli/list.ts`, `src/specialist/loader.ts`)
`SpecialistSummary` extended with `permission_required`, `interactive`, `thinking_level`, `skills` (paths array), `scripts` (phase/run/inject_output array). Renderer rewritten twice: first pass added fields as verbose multi-line blocks; second pass (this session) simplified to flat 2-space indent matching `sp --help` style — one header line (name | permission | keep-alive | thinking | model), one description line, optional skills line, optional scripts line.

### using-specialists SKILL.md (v3.7 → v3.10)
